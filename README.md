留言板
===

Python Web框架Tornado入门练习小项目：留言板

### 已经实现的功能：

- 用户的注册与登录，包括昵称，密码等合法性要求的检查
- 主要功能：用户登录后，进行留言
- 用户留言信息的显示：用户名，客户端ip，留言内容，留言时间
- 回复功能
- 留言数量过多，会自动分页显示功能
- 仪表盘，统计站点用户数量和留言数量

### 站点展示：

站点后台采用tornado，前台采用Amazon UI，数据库采用MongoDB
